# Dependencies

+ nodejs ^12.x.x
+ npm ^6.x.x

# Run On Development
  
  ```shell
  $ git clone https://github.com/winesports/affiliates
  $ cd affiliates
  $ npm install
  $ npm start
  ```
# Run On Production
  
  ```shell
  $ git clone https://github.com/winesports/affiliates
  $ cd affiliates
  $ npm install
  $ npm run build
  ```
